PHP API Kit v4.0, implementing v4.0 of the HandsetDetection API.

API Kits can use our web service or resolve detections locally depending
depending on your configuration.

API Kit configuration files can be downloaded directly from Handset Detection.

Login to your dashboard, click 'Add a new Site Profile' configure up your new
site then grab the config file variables for your API Kit.
